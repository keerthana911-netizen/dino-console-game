# 🦖 Dino Console Game (C++)

A Chrome Dino-style game built in C++ using Windows console.

## 🎮 Features
- Jump mechanics
- Moving obstacles 🌵
- Day/Night mode 🌙☀️
- Sound effects 🔊
- Score & High Score system
- Death effects 💀

## 🕹️ Controls
- Press SPACE to jump
